Library dockerfile-opam

This library exposes the following toplevel modules:

  • Dockerfile_distro Run Opam commands across a matrix of Docker containers. Each of these containers represents a different version of OCaml, Opam and an OS distribution (such as Debian or Alpine).
  • Dockerfile_linux Linux distribution-specific Dockerfile utility functions
  • Dockerfile_opam Rules for generating Dockerfiles involving OPAM.
  • Dockerfile_windows Windows specific Dockerfile utility functions
